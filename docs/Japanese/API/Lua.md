# Lua Script
この項目では、Luaスクリプトの主な使い方について説明します。
全てのAPIについての詳細は[こちら]をご覧ください。

## Luaって？
Luaは、1993年にブラジルで開発された、軽量･高速･C++やHaxeなどのC言語への組み込みを前提として設計された、

オープンソース(ソースコードが公開されている)のスクリプト言語。

ライブラリは、`linc_luagit`を使用しています。

このエンジンでは、下に記載されている拡張子を使ったスクリプトが使えます。

## 拡張子
* `.lua`
* `.luas`
* `.lscript`

  -- コードの記述
